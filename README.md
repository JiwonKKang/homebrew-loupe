# homebrew-loupe

Homebrew tap for [Loupe](https://github.com/JiwonKKang/loupe) — a human-first code review desktop app.

```bash
brew install --cask JiwonKKang/loupe/loupe
```

The build is ad-hoc signed (no Apple Developer account); the cask strips the
quarantine flag on install so it launches without a Gatekeeper prompt.
